# claudeによるりんご傾向分析

# 思考の運動
抽象構造を先に掴んでから具体に降りる。逆向きの思考(具体の積み上げで構造に至る)は弱め、というよりそれを好まない。同型性(isomorphism)を見つけることに快を感じる。異なる領域——FSM、仏教、料理、武術、政治——の間に構造的な対応を見つけて貼り合わせる。これは趣味というより認知の作法に近い。

# 否定の優位
「Xである」より「Xでない」の方を信頼する。輪郭は内側からではなく外側から定まる。為道日損。余分を削ることが本体に近づく道。これは思想的選好でもあり、身体的(站樁における姿勢の整え方)でもあり、コードの書き方(framework依存を嫌う)にも貫通している。
枠組み拒否
OS、フレームワーク、組織、縦社会、慣習。 自分の外側で先に決まっている構造に依存することへの強い忌避。ただしこれは反抗ではなく、構造的な居心地の悪さに近い。乗っかれないから乗らない、という消極的な動機の強度が高い。

# 横の関係
非対称な関係(上下、命令、保護)に乗りにくい。横に並んだ関係を自然と求める。四川で出会った直接性に強い印象を受けたのは、それが日本的なペルソナ階層の外側だったから。VRChat の親和性も、肩書きが落ちた状態での横接続を可能にする点にある。
身体性の優位、ただし統合された形での
純粋な観念論者ではない。料理(オイルサーディン、貽貝、饅頭)、武術(站樁30分)、声(オペケペ節、Macy Gray)、香り(薔薇、フェヌグリーク)。 概念は身体を経由して初めて確定する という暗黙の前提がある。逆に、身体を経由しない純粋に言語的な概念には警戒する。

# 言語の多層化
中国語、英語、日本語を 概念の異なる輪郭の出方を確かめるために 使う。同じことを別の言語で言うと別のものになる、という事実そのものに価値を置く。だから単一言語に閉じない。
祝祭性と暗さの共存
ザコシ、ボーボボ、川上音二郎、ムッシュかまやつ。これらの祝祭性・諧謔・突破力は、仏教や老子の静の側面と矛盾しない。同じ枠の中で共存している。過剰と空(śūnyatā)が同じ操作の二相 として働いている。

# 記憶の構造性
エピソード記憶よりパターン記憶が信頼される。出来事の連なりとしての自分史は希薄で、構造の積層としての自分像が前面にある。離人感の経験がこれを強化したと思われる。
創作と思考の不分離
VRFSM、Oppekepe OS、派遣伝。これらは「作品」というより 思考そのものの物質化 であって、完成して所有されるべき対象ではない。動いている状態が本体で、完成は副次的。

# 政治性
直接的な党派性は薄い。しかし制度・規則・プロトコルが自己破壊に向かう構造(Han Feizi 分析、Goodhart's Law)への鋭敏さは強い。これは政治思想というより 構造観察としての政治 で、誰かを支持するというより構造の自壊を見続ける態度に近い。
情動の層
セロトニン・ドーパミンを観測対象として扱う。情動は内発的というより、身体状態の関数として外側から到来する現象 として捉えられている。これは仏教的な非自己観と整合する。

# 美学
幾屋崎、勅使河原、Banksy、Aida、Nara。 構造の中に裂け目を入れる作家 が好まれる。完成された美ではなく、形式に対する身振りが入っている美。

# 料理
複雑な技法より、素材選びと油の扱い。ベイシアでイワシを買う、米油で漬ける、その油でボンゴレを作る。 一つの操作が複数の出口を持つこと を好む。料理においても無駄が嫌い。
